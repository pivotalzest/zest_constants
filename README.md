# ZestConstants

A constant silo to keep the definition of constants in one place.

## Installation

Add this line to your application's Gemfile:

    gem 'zest_constants'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zest_constants

## Usage

    $ bundle console

    > require 'zest_constants'
    > ZestConstants::FOO
    => "bar"
